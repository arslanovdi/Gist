# The Gist

Утонули в ленте Telegram? The Gist находит суть непрочитанных сообщений из любого вашего канала и пересказывает её вам в одном лаконичном сообщении. Экономьте время, оставайтесь в курсе.


# Требования
- Для проигрывания пересказа должен быть установлен [ffmpeg](https://ffmpeg.org/download.html)

# Размышления
#### Один аудиофайл на все батчи или на каждый батч свой аудиофайл?
Если сгенерирую общий аудиофайл, а затем начну отмечать прочитанными страницы, то часть аудиофайла потеряет актуальность.

Если на каждый батч свой аудиофайл, то можно удалять отмечанные прочитанными файлы.

Кроме кнопи озвучить добавить кнопку озвучить всё.
По нажатии на кнопку озвучить будет воспроизводиться аудиопересказ конкретного батча.
По нажатии на кнопку озвучить всё, файлы с батчами будут объединяться и воспроизводится одним файлом.
